Step 1 — Create a Free Firebase Project

Firebase (by Google) is a free cloud database that stores all BPMC lab data permanently and syncs instantly across every device — no server needed, no monthly cost.

📋 Instructions (≈ 3 minutes)
  1. Go to console.firebase.google.com — sign in with any Google account
  2. Click "Add project" → enter any name (e.g. bpmc-lab) → Continue → Create Project
  3. In your project dashboard, click "Build" in the left menu → "Realtime Database"
  4. Click "Create Database" → choose location: Singapore (asia-southeast1) → Next
  5. Choose "Start in test mode" → Enable
  6. Click the ⚙️ gear icon (top-left) → "Project settings"
  7. Scroll down to "Your apps" → click the </> Web icon → enter any nickname → Register app
  8. You will see a code block with firebaseConfig = { ... } — copy the entire block
✅ 100% Free. Firebase's Spark plan includes 1 GB storage, 10 GB/month transfer, and 100 simultaneous users — far more than any hospital lab needs.

Step 2 — Paste Your Firebase Config

Paste the firebaseConfig JSON you copied from Firebase. You can paste it with or without the const firebaseConfig = wrapper.

// It will look like this (your values will differ):
const firebaseConfig = {
  apiKey: "AIzaSyABC123...",
  authDomain: "bpmc-lab.firebaseapp.com",
  databaseURL: "https://bpmc-lab-default-rtdb.asia-southeast1.firebasedatabase.app",
  projectId: "bpmc-lab",
  storageBucket: "bpmc-lab.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123:web:abc"
};
